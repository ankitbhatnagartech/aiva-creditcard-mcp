API Action Reference

query_or_actionRequired Extra FieldsDescriptionFlow
get_transactionsaccount_number, card_numberReturns last 5-10 transactions + card summaryAll flows (Step A)
select_transaction+ transaction_id (or omit if blocked)Details + probable action; or security question if blockedAll flows (Step B)
request_limit_increase+ transaction_id (failed txn)Increases daily credit limit to $2,000, logs actionFlow 1
unblock_card+ security_answerValidates security answer; sets card Active, logs actionFlow 2
confirm_fraud+ transaction_idBlocks card, marks txn disputed, orders replacement, logsFlow 3a
deny_fraud+ transaction_idClears fraud flag, card stays Active, logs actionFlow 3b
